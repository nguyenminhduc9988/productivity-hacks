What this curve shows. Reading the curve at x = X tells you the expected score per game when the method is run at an average per-game budget of X tokens. Slow games may use more, fast games less; the average works out to X.

What-if: per-level token caps

Each curve shows what the score-vs-tokens curve would have been if every level had been stopped at the listed per-level token cap. The cap is on cumulative tokens spent since the most recent level win: cross the cap without winning the level and the lane truncates there with no further credit (full credit is kept for previously-won levels). 100% is the largest token total any single level actually consumed in this run (won or not); the other curves are fractional cuts of that.
